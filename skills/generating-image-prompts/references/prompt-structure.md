# Prompt Structure

Detailed anatomy of effective prompts for AI image generation.

## Core Elements

| Element | Description | Examples |
|---------|-------------|----------|
| **Subject** | Who/what is in the image | living room, kitchen, modern house, floor plan |
| **Composition** | How the shot is framed | wide shot, close-up, birds-eye view, top-down view |
| **Environment** | Where the scene takes place | indoors, outdoor terrace, garden, urban setting |
| **Style** | Overall aesthetic | Scandinavian, minimalist, contemporary, rustic |

## Advanced Elements

| Element | Description | Examples |
|---------|-------------|----------|
| **Lighting** | Type of light | natural daylight, golden hour, soft ambient, dramatic shadows |
| **Camera** | Technical settings | shallow depth of field, wide angle, f/1.8, eye-level |
| **Color** | Palette and tones | warm tones, muted palette, neutral colors, high contrast |
| **Mood** | Feeling to evoke | cozy, serene, energetic, calm, inviting |
| **Medium** | Visual style | photo, 3D render, watercolor, architectural drawing |

## Aspect Ratios

| Ratio | Use Case |
|-------|----------|
| 16:9 | Landscape, wide interiors, exteriors |
| 9:16 | Vertical, mobile, tall spaces |
| 1:1 | Square, floor plans, social media |
| 4:3 | Traditional photo, balanced compositions |
| 21:9 | Cinematic, panoramic views |
| 3:2 | Classic photography ratio |

## Lighting Vocabulary

### Natural Light

- natural daylight
- golden hour (warm, low sun)
- blue hour (cool, twilight)
- overcast soft light
- dappled sunlight through trees
- bright midday sun

### Artificial Light

- ambient lighting
- soft diffused light
- dramatic shadows
- warm pendant lights
- recessed LED lighting
- studio lighting

## Composition Terms

### Camera Angles

- eye-level view
- low angle (looking up)
- high angle (looking down)
- birds-eye view (directly above)
- worms-eye view (from ground)
- dutch angle (tilted)

### Shot Types

- wide shot (full room/building)
- medium shot (partial view)
- close-up (detail focus)
- extreme close-up (texture/material)
- establishing shot (context)

### Framing

- symmetrical composition
- rule of thirds
- leading lines
- framed within frame
- negative space

## Color Palettes

| Term | Description |
|------|-------------|
| warm tones | oranges, yellows, reds, browns |
| cool tones | blues, greens, grays |
| neutral palette | whites, beiges, grays, taupes |
| muted colors | desaturated, soft tones |
| vibrant | saturated, bold colors |

| earthy tones | browns, terracottas, olives |
