# Architecture Vocabulary

Vocabulary for exterior architecture and floor plan prompts.

## Architectural Styles

| Style | Characteristics |
|-------|-----------------|
| Modern | clean lines, minimal ornamentation, open spaces |
| Contemporary | current trends, mixed materials, large windows |
| Scandinavian | simplicity, functionality, natural materials |
| Minimalist | extreme simplicity, monochromatic, essential forms |
| Swiss chalet | timber, pitched roof, balconies, mountain context |
| Mediterranean | stucco, terracotta, arches, courtyards |
| Industrial | exposed structure, metal, concrete, raw materials |
| Brutalist | raw concrete, geometric, bold forms |
| Mid-century modern | organic forms, large windows, integration with nature |
| Japanese | zen, natural materials, sliding doors, simplicity |
